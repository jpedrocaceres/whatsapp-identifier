Privacy Policy / Política de Privacidade

WhatsApp Identifier — Chrome Extension

1. Visão Geral

Esta Política de Privacidade descreve como a extensão WhatsApp Identifier coleta, usa e protege as informações dos usuários. O desenvolvedor desta extensão é João Pedro Cáceres, pessoa física.

Esta extensão foi projetada com privacidade por padrão: não coleta, armazena nem transmite qualquer dado pessoal para servidores externos.

2. Dados Coletados Nenhum dado externo

Esta extensão não coleta nenhum dos seguintes tipos de dados:

  • Informações de identificação pessoal
  • Conteúdo de mensagens do WhatsApp
  • Histórico de navegação
  • Dados de localização
  • Informações financeiras ou de saúde
  • Comunicações pessoais

3. Uso das Permissões

Permissão de host (web.whatsapp.com): Utilizada exclusivamente para injetar um script de conteúdo na interface do WhatsApp Web, permitindo detectar e modificar o campo de texto de mensagens para prefixar automaticamente o nome configurado pelo usuário.

Permissão tabs: Utilizada para recarregar a aba do WhatsApp Web quando necessário e sincronizar o estado da extensão com a aba ativa. Nenhuma informação sobre outras abas ou sites é acessada.

Permissão storage: Utilizada para salvar localmente as preferências do usuário (nome/prefixo e configurações de privacidade) via chrome.storage.local. Os dados ficam armazenados apenas no dispositivo do usuário e nunca são enviados a servidores externos.

4. Funcionalidade Privacy Blur

A extensão oferece uma funcionalidade opcional de desfoque de privacidade que aplica um filtro CSS sobre nomes e mensagens na interface do WhatsApp Web. Este recurso:

  • Opera inteiramente no navegador do usuário, sem enviar dados a servidores
  • Pode ser ativado ou desativado a qualquer momento pelo usuário
  • Não altera, intercepta ou armazena o conteúdo das mensagens

5. Armazenamento de Dados

Os únicos dados armazenados pela extensão são as preferências de configuração do usuário (nome/prefixo e estado do Privacy Blur), salvos localmente no navegador Chrome. Esses dados:

  • Permanecem exclusivamente no dispositivo do usuário
  • Não são transmitidos a nenhum servidor
  • Podem ser removidos a qualquer momento desinstalando a extensão

6. Compartilhamento de Dados

Nenhum dado é compartilhado com terceiros. A extensão não utiliza serviços de análise, rastreamento, publicidade ou qualquer outro serviço externo.

7. Alterações a Esta Política

Eventuais atualizações a esta política serão publicadas nesta mesma página. Recomendamos revisar esta política periodicamente.

8. Contato

Para dúvidas sobre esta Política de Privacidade, entre em contato pelo e-mail cadastrado na conta de desenvolvedor Google associada a esta extensão.

1. Overview

This Privacy Policy describes how the WhatsApp Identifier Chrome extension handles user information. The developer of this extension is João Pedro Cáceres, an individual developer.

This extension is built with privacy by default: it does not collect, store, or transmit any personal data to external servers.

2. Data Collected No external data

This extension does not collect any of the following:

  • Personally identifiable information
  • WhatsApp message content
  • Browsing history
  • Location data
  • Financial or health information
  • Personal communications

3. Permission Usage

Host permission (web.whatsapp.com): Used exclusively to inject a content script into the WhatsApp Web interface, enabling the extension to detect and modify the message input field in order to automatically prepend the user-configured name prefix.

tabs permission: Used to reload the WhatsApp Web tab when necessary and synchronize the extension state with the active tab. No information from other tabs or websites is accessed.

storage permission: Used to locally save the user's preferences (name/prefix and privacy settings) via chrome.storage.local. This data is stored only on the user's device and is never sent to any external server.

4. Privacy Blur Feature

The extension offers an optional privacy blur feature that applies a CSS blur filter over names and messages in the WhatsApp Web interface. This feature:

  • Operates entirely within the user's browser, without sending data to any server
  • Can be enabled or disabled at any time by the user
  • Does not alter, intercept, or store message content

5. Data Storage

The only data stored by this extension is the user's configuration preferences (name/prefix and Privacy Blur state), saved locally in the Chrome browser. This data:

  • Remains exclusively on the user's device
  • Is never transmitted to any server
  • Can be removed at any time by uninstalling the extension

6. Data Sharing

No data is shared with third parties. The extension does not use analytics, tracking, advertising, or any other external service.

7. Changes to This Policy

Any updates to this policy will be published on this same page. We recommend reviewing this policy periodically.

8. Contact

For questions about this Privacy Policy, please contact us via the email address registered on the Google developer account associated with this extension.
